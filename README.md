# Checkout

Simple interface to register baskets.

## Quick start
Clone, then
```shell
uv sync
uv run pre-commit install --install-hooks
inv test
uv run ./src/manage.py migrate
uv run ./src/manage.py generate_dummy_baskets
uv run ./src/manage.py createsuperuser
```
